Projects

PokeDex

The app connects to the PokeApi to deliver dynamic information about any Pokémon on request. It retrieves and showcases Pokémon’s Name, Cry (with playback functionality on button click), Image, Type, Abilities, Flavor Text, ID, Height, Weight, Base Stats, Gender Rates, and Evolution Chain. Interactive charts for Base Stats and Gender Rates are created using Charts.js. This is the second version of the application.

Discord.py Bot

A bot for the Discord using discord.py wrapper. Some commands but not all:
- User commands: 8ball, guess, spam, dog, and help. The animals commands returns an image, video or gif of said animal, and is api based.
- Dev commands: load, unload, and reload. These commands are adjust extensions without bot downtime.
- Mod Features: the ability to track deleted and edited messages. These are server side operations for the selected few.

Pixel-Ascii Art

Sometimes life gives you lemons but instead of taking that lemon, you decide to grow a lemon tree, harvest the lemons and then make lemonade. Rather than using a converter, I made one. This project uses the p5.js to outline a canvas, populated it with an image, creates a pixel mapping of the image and colors it. To create the Ascii masking, I grayscale the pixels and mapped them to characters based on the intensity of the gray.
