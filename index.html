Projects

PokeDex

The app connects to the PokeApi to deliver dynamic information about any Pokémon on request. It retrieves and showcases Pokémon's Name, Cry (with playback functionality on button click), Image, Type, Abilities, Flavor Text, ID, Height, Weight, Base Stats, Gender Rates, and Evolution Chain. Interactive charts for Base Stats and Gender Rates are created using Charts.js. This is the second version of the application.

Discord.py Bot

A bot for the Discord using discord.py wrapper. Some commands:
- User commands: 8ball, guess, spam, dog, and help. The dog command returns an image, video or gif of said animal.
- Dev commands: Load, unload, and reload. These commands are use to adjust extensions without bot downtime.
- Mod Features: The ability to track deleted and edited messages. These are server side operations that need prior setup.

Pixel-Ascii Art

Sometimes life gives you lemons but instead of taking that lemon, you decide to grow a lemon tree, harvest the lemons and then make lemonade. Rather than using a converter, I made one. This project uses the p5.js to outline a canvas, populated it with an image, creates a pixel mapping of the image and colors it. To create the Ascii masking, I grayscale the pixels and mapped them to characters based on the intensity of the gray.

NYC Prediction Model

A prediction model for properties in Manhattan, the Bronx, Brooklyn, Queens, and Staten Island, using data from NYC databases. Property sales and pricing are categorized by borough, neighborhood, and housing type: one-family, two-family, and three-family homes. The model aims to predict the pricing of each housing type within a given borough and its neighborhoods using linear regression. Predictions are then visualized on a map using GeoMaps.
